# Abraham Teklu - Personal Website

Personal website for job applications and displaying interests. Deployed via GitHub Pages.

## Setup Instructions

1. **Add Images**
   - Create an `images` folder in the root directory
   - Add the following images:
     - `hero-image.jpg` - Main hero image with group photo
     - `neutron-reconstruction.jpg` - Project image for Neutron Reconstruction
     - `t2k-nova-analysis.jpg` - Project image for T2K-NOvA Analysis
     - `sfgd-construction.jpg` - Project image for SFGD Construction
     - `scintillator-prototypes.jpg` - Project image for Scintillator Prototypes
     - `contact-notes.jpg` - Image for contact page (pinned notes)
     - `inspire-logo.png` - INSPIRE HEP logo
     - `researchgate-logo.png` - ResearchGate logo
     - `linkedin-logo.png` - LinkedIn logo

2. **Update Links**
   - Update the profile links in `index.html` (INSPIRE HEP, ResearchGate, LinkedIn)
   - Update project paper links in the Novel 3D Scintillator project section

3. **Deploy to GitHub Pages**
   - Push this repository to GitHub
   - Go to Settings > Pages
   - Select the main branch as the source
   - Your site will be available at `https://[username].github.io/[repository-name]`

## Features

- Responsive design
- Scroll animations for featured projects
- Contact form
- Clean, professional layout matching Squarespace design

## Pages

- **Home** (`index.html`) - Hero section and featured research projects
- **Resume** (`resume.html`) - Resume page (add your resume content)
- **Interests** (`interests.html`) - Interests page (add your interests)
- **Contact** (`contact.html`) - Contact form and information
